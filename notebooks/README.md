# Notebooks

This directory contains all Jupyter notebooks organized by course.

## Structure

### DPP (Distributed Parallel Processing)
- `00-Introduction/` through `11-Chapter-11/`: Chapter-based learning notebooks
- `Practicals/`: Practical exercise notebooks
- `Exercises/`: Personal practice and exercises
- `Notes/`: Additional notes and references

### ECP (English for Career Preparation)
- `01-Introduction-to-Proposals/`: Basics of proposal writing
- `02-Business-Proposal-Format/`: Format and structure
- `03-Sample-Analysis/`: Analysis of sample proposals
- `04-Practice-Proposals/`: Practice writing proposals

## Usage

Each chapter folder should contain:
- Main learning notebook (e.g., `chapter-01-overview.ipynb`)
- Code examples (if applicable)
- Exercises and solutions
- Summary notes

## Naming Convention

- Use descriptive names with hyphens: `topic-name.ipynb`
- Number notebooks if order matters: `01-introduction.ipynb`, `02-concepts.ipynb`
- Keep names lowercase for consistency
