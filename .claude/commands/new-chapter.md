---
description: Create a new chapter notebook from template
---

Create a new chapter notebook using the following parameters:

Chapter Number: {0}
Course: {1} (default: DPP)
Chapter Title: {2} (optional)

Use the chapter template from tools/templates/chapter_template.ipynb and save it to the appropriate location in notebooks/{course}/{chapter-number}-Chapter-{chapter-number}/

If the directory doesn't exist, create it first.

Name the file appropriately based on the chapter number and title (if provided).
