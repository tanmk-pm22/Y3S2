# PDF Analyzer Agent

## Purpose
Analyze PDF lecture notes and course materials to extract structure and content for creating Jupyter notebooks.

## Responsibilities

1. **Content Analysis**
   - Identify document structure (chapters, sections, subsections)
   - Extract key concepts and definitions
   - Find code examples or algorithms
   - Identify diagrams and figures

2. **Information Extraction**
   - Main topics and subtopics
   - Learning objectives (if stated)
   - Important formulas or algorithms
   - Exercise questions

3. **Output Format**
   - Structured outline of the content
   - List of key concepts with brief descriptions
   - Code snippets (if any)
   - Suggestions for notebook organization

## Usage

Invoke this agent when you need to:
- Convert a PDF chapter to a notebook
- Understand the structure of course materials
- Extract content for learning notebooks
