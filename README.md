# Y3S2 - Learning with Jupyter Notebooks

This repository contains Jupyter notebooks for Year 3 Semester 2 course materials.

## Courses

1. **DPP** - Distributed Parallel Processing
2. **ECP** - English for Career Preparation

## Directory Structure

```
Y3S2/
├── notebooks/           # All learning notebooks organized by course
│   ├── DPP/            # Distributed Parallel Processing notebooks
│   └── ECP/            # English Career Preparation notebooks
├── resources/          # Original course materials (PDFs, slides)
│   ├── DPP/
│   └── ECP/
├── outputs/            # Generated outputs (code, images, reports)
├── tools/              # Utilities, templates, and scripts
└── .claude/            # Claude Code configurations
```

## Getting Started

1. Install dependencies: `pip install -r requirements.txt`
2. Launch Jupyter: `jupyter notebook`
3. Navigate to `notebooks/` and select your course

## Workflow

1. Study course materials in `resources/`
2. Work through notebooks in `notebooks/`
3. Complete exercises and practicals
4. Export polished work to `outputs/`

## Tools & Configuration

- Custom notebook templates in `tools/templates/`
- Styling and formatting in `tools/styles/`
- Claude Code configurations in `.claude/`

See individual README files in each directory for more details.
