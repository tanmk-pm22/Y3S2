# Distributed Parallel Processing

Course materials and Jupyter notebooks for learning distributed and parallel processing concepts.

## Topics Covered

### 01 - Introduction
- Fundamentals of parallel processing
- Parallel vs distributed computing
- Performance metrics and Amdahl's Law

### 02 - Parallel Architectures
- Shared memory vs distributed memory
- SIMD, MIMD architectures
- GPU computing basics

### 03 - Threading and Multiprocessing
- Python threading and multiprocessing
- Synchronization mechanisms
- Race conditions and deadlocks

### 04 - Distributed Computing
- Distributed system fundamentals
- CAP theorem
- Consistency models

### 05 - Message Passing
- MPI (Message Passing Interface)
- Point-to-point communication
- Collective operations

### 06 - MapReduce and Spark
- MapReduce programming model
- Apache Spark basics
- RDDs and DataFrames

## Directories

- `exercises/` - Practice problems and solutions
- `projects/` - Course projects and assignments
- `resources/` - Additional materials, papers, and references

## Getting Started

1. Install required dependencies: `pip install -r ../requirements.txt`
2. Start with notebooks in `01-introduction/`
3. Progress through topics sequentially
